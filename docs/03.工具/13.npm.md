npm的一些设置和命令

```shell

# 查询一下当前 NPM 源：
npm get registry  # 一般返回的应该都是： https://registry.npmjs.org/，这是默认的


# 将 NPM 源设置为国内的淘宝镜像源
npm config set registry http://registry.npm.taobao.org/


# 换回原来的源，只需要执行以下指令即可：
npm config set registry https://registry.npmjs.org/


```
