### 设置ZSH终端走代理

```shell
code ~/ .zshrc

# 设置使用代理
alias setproxy="export http_proxy=http://127.0.0.1:1080; export https_proxy=$http_proxy; export all_proxy=socks5://127.0.0.1:1080; echo 'Set proxy successfully'"
# 设置取消使用代理
alias unsetproxy="unset http_proxy; unset https_proxy; unset all_proxy; echo 'Unset proxy successfully'"

source ~/ .zshrc

```
