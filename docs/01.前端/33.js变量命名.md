### 1.1 布尔类变量的命名前缀补充

![](https://qiniu.espe.work/blog/20220127110613.png)

has 常用于表示有没有或者是否**包含** / 而 is 常用于表示**是不是，是否**

has 例：

```typescript
const hasLocationPermission = ?
// 有没有定位权限
const hasUserInfo = ?
// 有没有用户信息
```

is 例：

```typescript
const isShow = ?
// 是否（展示/显示)
const isTiemout = ?
// 是否超时
```
