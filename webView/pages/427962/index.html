svg和canvas的优缺点

# Svg 和 Canvas 的优缺点

  1. Svg
  • 使用 XML 文档描述来绘图
  • 可缩放矢量图形
  • 支持事件处理器
  1. Canvas
  • 使用 JavaScript 程序绘图
  • 能够与浏览器渲染引擎紧密结合、节约资源，简化了图形和网页中其他元素的交互过程
  • 操作普通 HTML 元素一样操作 canvas
  • 降低了维护与更新的难度
  • 效率比Svg高
  • 不支持事件处理器
